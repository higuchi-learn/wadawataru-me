## 画像アップロード（R2 連携）

- ブランチ: 未作成
- 画像をペーストしたら Cloudflare R2 バケットにアップロードし、アクセス URL を返す
- 返ってきた URL をサムネイル入力欄または Markdown エディタの本文に埋め込む

### 実装に必要なもの

**バックエンド**
- R2 バケットの設定（`wrangler.toml` へのバインディング追加）
- 画像受け取り・R2 アップロード・URL 返却の API ルート（`/api/upload`）が必要
  - バイナリデータの送受信になるため Server Action ではなく API ルートが適している
  - Cloudflare Pages 上では R2 のネイティブバインディング（`env.R2_BUCKET`）が使える

**フロントエンド（BlogEditor）**
- サムネイル入力欄: `onPaste` イベントでクリップボードの画像を取得 → `/api/upload` に送信 → 返ってきた URL をセット
- Markdown エディタ: SimpleMDE の `onPaste` または `drop` イベントをフックして同様の処理を行い、`![](URL)` の形式で本文に挿入
